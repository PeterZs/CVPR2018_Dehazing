## Dependency
* For Running
    * Python 2
    * keras
    * scipy
	* numpy
	* scikit-image
	* matplotlib

## Running
```
$ cd src/
$ python main.py  <hazy_image_dir>   <output_dir>
```
This runs the code in the supplied images.
```
$python main.py  ../data/hazy_img/ ../data/out/
```
